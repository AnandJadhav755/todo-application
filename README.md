# TaskFlow - Todo Application

A Spring Boot Todo Application built using:

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- HTML/CSS

## Features

- Add Task
- Update Task
- Delete Task
- Mark Task as Completed
- View All Tasks
